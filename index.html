☁️ Google Sheets Sync
Connect this app to a shared Google Sheet so your whole team sees the same live data — WIR, NCR, RFI, SNA, QVN — updated in real time.
📋 Step-by-Step Setup
1
Go to sheets.google.com → Create a new spreadsheet → Name it HSR QMS – TrackWay
2
Create 5 tabs named exactly: RFI  NCR  SNA  QVN  WIR
3
Copy the Sheet ID from the URL:
docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit
4
Go to console.cloud.google.com → Create API Key → Enable Google Sheets API → Restrict key to Sheets API only
5
Set the sheet Share to "Anyone with the link can view" (read). For write sync, also set up an Apps Script (see below).

📝 Apps Script (for write sync)
1
In your Google Sheet → Extensions → Apps Script → paste this code and deploy as Web App (Anyone can access):

function doPost(e){var d=JSON.parse(e.postData.contents);var ss=SpreadsheetApp.openById(d.sheetId);var sh=ss.getSheetByName(d.tab);sh.clearContents();sh.getRange(1,1,d.rows.length,d.rows[0].length).setValues(d.rows);return ContentService.createTextOutput("ok");}
